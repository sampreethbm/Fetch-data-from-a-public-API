# Weather App Using JavaScript

## Project Description

This is a simple weather application created using
HTML, CSS, and JavaScript.

The app uses a free public API (Open-Meteo) to get
real-time weather information.

The user enters a city name and the app shows
temperature and wind speed.

---

## Features

- Uses Fetch API with async/await
- Gets live weather data
- Shows output on the screen
- Handles errors using try-catch
- Simple and easy user interface

---

## Technologies Used

- HTML (for structure)
- CSS (for design)
- JavaScript (for logic)
- Open-Meteo API (for weather data)

---

## How It Works

1. User enters a city name.
2. JavaScript converts the city name into latitude and longitude.
3. Weather data is fetched using API.
4. Data is displayed on the screen.
5. If any error occurs, an error message is shown.

## How To Run The Project

1. Download or copy the project folder.
2. Open the folder.
3. Double click on index.html.
4. The app will open in your browser.
5. Enter a city name and click "Get Weather".
